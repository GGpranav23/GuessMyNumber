git branch - List all branches

git checkout -b main - Create and switch to new branch 'main'

git branch -d master - Delete the master branch

git remote -v - Shows all remote repositories and their URLs (fetch and push)

git config --get user.email - Shows the configured Git email address used for commits


git config --get user.name - Shows the configured Git username used for commits

git config --global user.email "your.email@example.com" - Sets the Git email address for all repositories

git config --global --unset-all user.name - Removes all globally set Git usernames from your configuration (useful when duplicate or conflicting values exist).
